题目二：最长公共子串

原题

题目描述
给定两个字符串，请你找出它们最长的公共子串的长度。（子串必须是连续的）

思路：

经过实验，发现本题可以暴力枚举，只要将两个for循环嵌套把两个字符串的字符挨个比对，如果对上则求得在此时的公共子串长即可

由此即联想到了这种方法太过麻烦，当 str1[i] 与 str2[j] 相等时，他们的长度应该可以是1 + str1[i + 1] 与 str2[j + 1] 的公共子串长。那么就相当于动态规划的记忆化搜索，所以本题可以用动态规划求解。但如果我们正序遍历，当要求 str1[i] 和 str2[j] 时，str1[i + 1] 和 str2[j + 1] 仍不清楚，所以不妨设dp [i] [j] 为以 str1[i] 和 str2[j] 字符为结尾的公共子串长

dp [i] [j] 可能有以下两种情况：

  1. str1[i] == str2[j] 此时 dp [i] [j] = 1 + dp [i - 1] [j - 1] (状态转移方程)
  2. str1[i] != str2[j] 此时 dp [i] [j] = 0

在遍历的同时求出最值即可

题目三：最长回文子串

原题

给定一个字符串 s，找到 s 中最长的回文子串，假定唯一。 力扣原题点这里哦

​

思路：

首先经实验这道题可以暴出来，仍然是双指针从头尾向中间遍历，如果相等就判断是否回文T^T

但反向思考，判断是否回文同样也可以是从中心向两边遍历，由此引出了第一个方法：中心扩展算法

即遍历字符串，在每一个字符处尝试扩展得到以其为中心的最长回文子串。但字符数为奇/偶的回文串边界条件（最后会剩下一个或两个字符）不同，所以要分别判定

同样在扩展字符时发现其存在状态转移方程，如果设 dp [i] [j] 为 s 中下标从 i 到 j 的字符串，那么 dp [i] [j] 为回文串的条件为 dp [i +1] [j - 1] 为回文串且s[i] == s[j] 而边界条件就是仅剩下一个或两个字符（j - i <= 1）

题目五：统计单词数

原题

题目描述
一般的文本编辑器都有查找单词的功能，该功能可以快速定位特定单词在文章中的位置，有的还能统计出特定单词在文章中出现的次数。
现在，请你编程实现这一功能，具体要求是：给定一个单词，请你输出它在给定的文章中出现的次数和第一次出现的位置。注意：匹配单词时，不区分大小写，但要求完全匹配，即给定单词必须与文章中的某一独立单词在不区分大小写的情况下完全相同，如果给定单词仅是文章中某一单词的一部分则不算匹配。
输入格式
共 2 行。
第 1 行为一个字符串，其中只含字母，表示给定单词；
第 2 行为一个字符串，其中只可能包含字母和空格，表示给定的文章。

思路：

非常简单的题目，将单词和文章中字符全部转化为大写/小写，记录空格的位置，挨个比对空格中间的字符串和目标字符串，记录相等的第一次位置和总数

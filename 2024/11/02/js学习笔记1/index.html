变量

字母、下划线或美元$开头，后续可以有数字

声明变量：

1、使用var: var x = 1;用以声明局部和全局变量

3、使用let: let y = 1;声明块作用域局部变量

函数

函数闭包

闭包是 JavaScript 中最强大的特性之一。

JavaScript 允许函数嵌套

内部函数具有定义在外部函数中的所有变量和函数（以及外部函数能访问的所有变量和函数）的完全访问权限。

但是，外部函数却不能访问定义在内部函数中的变量和函数。这给内部函数的变量提供了一种封装。

由于内部函数可以访问外部函数的作用域，因此当内部函数生存周期大于外部函数时，外部函数中定义的变量和函数的生存周期将比内部函数执行的持续时间要长。当内部函数以某一种方式被任何一个外部函数之外的任何作用域访问时，就会创建闭包。

argument对象

函数的实际参数会被保存在一个类似数组（并不是数组，不包括数组的那些操作）的 arguments 对象中。

以此访问传入的参数，从0开始

数量由arguments.length表示

由此可以在函数中调用比声明时更多的参数（对应的你可以传比声明定义时的参数数更多的参数）

默认参数

以上函数在只接收了了一个参数时不会报错，可以视作其中每个参数声明时都有(b = undefined)

同样这个默认值是可以自行更改的比如（b = 1）

剩余参数

将不确定数量的参数表示为数组

此处multiply 函数使用剩余参数收集从第二个参数开始到最后的参数，然后该函数将它们与第一个参数相乘。

箭头函数

= >

具有较短的语法且没有它自己的 this、arguments、super和 new.target

箭头函数总是匿名的

由于没有自己的this，在闭包中this指向的是外函数的this

预定义函数

eval()：计算以字符串表示的js代码

isFinite():判断传入的值是否是有限的数值

isNaN():判断一个值是否是NaN

parseFloat()/parseInt()解析字符串参数，并返回浮点/整数

数组

创建数组

1、arr = new Array(数组元素)/（数组长度）：创建一个长度不为0但没有元素的数组

2、arr = Array(数组元素)/（数组长度）

3、arr = [数组元素]/[ ]

4、arr = Array.of(单个元素)

具有length属性

当使用单个数字初始化时只能使用括号语法，否则会混淆

不管如何创建数组，JS中的数组实际上都属于Array对象的实例

基本方法

arr[x]可以填充数组或更改数值元素

如果x为非整型，那么将作为数组的对象的属性

理解

实现上，JS是将数组作为对象，元素作为属性，数组索引与length作为属性名存储

自己给length赋值可以更改数组的大小

所有元素通过形参传入的字符连接成新字符串:join()

添加元素并返回length:push()

对象

概述

对象是一系列属性的集合，一个属性包含一个名和一个值

当属性的值是函数时，属性也会被称为“方法”

属性

可以通过 . 或者[ ]访问对象的属性(JS中只能使用String类型作为键类型)

[ ]中写的是属性名的字符串常量，要加“ ”

当然也可以不加括号，以下是它们的区别：

不加引号：如果你使用的是有效的标识符（例如字母、数字（不能以数字开头）、下划线或美元符号），可以不加引号

但在此时，如果没有预先定义key = ‘name’，那么属性名将是’undefined’

加引号：如果属性名包含空格、特殊字符，或者以数字开头，你需要加引号。

for…in

此处key相当于对象中的属性名，所以用[ ]访问

创建对象

可以通过对象初始化器或者构造函数和new来初始化对象

注意这其中engine也是一个对象

使用构造函数

  1. 通过创建一个构造函数来定义对象的类型（声明类型的名称 属性和方法），通常首字母大写
  2. 通过 new 创建对象实例

如果想在对象中嵌套一个对象，直接把要嵌套的对象作为参数传入就行（因为没有对参数类型的判断），访问被嵌套对象的属性就可以是对象名A.被嵌套对象名B.B中属性名

Object.create()方法

它允许你为创建的对象选择一个原型对象，而不用定义构造函数。

为之前创建的对象增加属性

通过prototype属性，可以为该类型所有对象而非一个对象增加了一个属性

this

JS中this的使用方式相似，在一个方法中指被调用的对象

当与 form 属性一起使用时，this可以指代当前对象的父窗体。

getter与setter

getter可以获取某个特定属性的值

setter可以设定某个属性

这俩都可以：

  • 使用 使用对象初始化器定义
  • 也可以之后随时使用 getter 和 setter 添加方法添加到任何对象

对象初始化器：在getter前加get（必须无参），在setter前加set（只接受一个参数）

添加getter/setter:Object.defineProperties方法，这个方法的第一个参数是你想定义 getter 或 setter 方法的对象，第二个参数是一个对象，这个对象的属性名用作 getter 或 setter 的名字，属性名对应的属性值用作定义 getter 或 setter 方法的函数

删除属性

delete可以删除一个属性

比较对象

在 JavaScript 中 objects 是一种引用类型。两个独立声明的对象永远也不会相等，即使他们有相同的属性，只有在比较一个对象和这个对象的引用时，才会返回 true.

静态属性

computed()计算属性

它可以让我们创建一个计算属性 ref，这个 ref 会动态地根据其他响应式数据源来计算其 .value

响应式数据：指通过 Vue 提供的机制，使得数据的变化能够自动反映到视图（UI）上。当在 Vue 组件中定义数据并将其标记为响应式后，Vue 会监控这些数据的变化，并在数据变化时自动更新相关的 DOM。创建方法：

1、ref:用于创建单一的响应式值，适用于基本类型（如字符串、数字）

2、reactive:用于创建响应式对象，适用于复杂数据结构（如对象、数组）

重新回到computed，它会自动追踪它所依赖的响应式数据。你在 computed 中使用的任何响应式属性，都会被 Vue 追踪并建立依赖关系。
